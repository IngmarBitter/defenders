If God does not exist,
=> there is no moral standard binding for all humans
=> objective moral values and duties do not exist. 2. Objective moral values and duties do exist. 3. => Therefore, God exists. Premise 1. is logical equivalent to:
1b. If objective moral values duties do exist,
=> there is a moral standard binding for all humans
=> God does exist and His inherent goodness is that standard.
